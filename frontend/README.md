# Frontend — Alfa Invoice Orchestrator

Фронтенд-часть проекта **Alfa Invoice Orchestrator**, разрабатываемого в рамках проектного практикума УрФУ командой **RTF Wizards**.

## Описание

Данный модуль представляет собой пользовательский интерфейс системы автоматизации формирования счетов-фактур по проводкам АБС.

На текущем этапе фронтенд ориентирован на реализацию базового интерфейса для работы с большими объемами данных, включая:

- просмотр реестров;
- отображение связанных сущностей;
- применение фильтров;
- оптимизированную загрузку данных;
- работу с большими выборками записей без полной загрузки всего объема на клиент.

## Задача фронтенда

Одной из ключевых задач является построение интерфейса, способного эффективно работать с очень большими наборами данных — вплоть до миллионов записей.

В связи с этим в проекте особое внимание уделяется:

- ленивой загрузке данных;
- постраничному получению записей;
- оптимизации рендера;
- снижению нагрузки на браузер;
- удобной работе пользователя с таблицами, фильтрами и списками.

## Запуск проекта

Перейдите в директорию `frontend`:

```bash
cd frontend
```

Установите зависимости:

```bash
npm i
```

Запустите проект в режиме разработки:

```bash
npm run dev
```
