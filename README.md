# Alfa Invoice Orchestrator

**MVP учетной системы для автоматизации формирования счетов-фактур по проводкам АБС**  
Проектный практикум УрФУ, команда **RTF Wizards**

## О проекте

Данный репозиторий содержит проект, разрабатываемый в рамках проектного практикума УрФУ совместно с заказчиком.

Цель проекта — создать минимально жизнеспособный продукт (MVP) учетной системы, предназначенной для автоматизации формирования счетов-фактур на основе проводок, поступающих из Автоматизированной Банковской Системы (АБС).

Система ориентирована на обработку доходных и НДС-проводок, их связывание по заданным атрибутам, формирование проектов счетов-фактур, а также предоставление пользовательского интерфейса для просмотра, редактирования и дальнейшей отправки данных в другую учетную систему.

## Актуальность

В рамках банковских процессов формирование счетов-фактур на основании проводок является важной частью учета операций по продаже услуг. При большом объеме данных ручная обработка таких операций становится трудоемкой, медленной и подверженной ошибкам.

Разрабатываемая система должна сократить объем ручной работы, повысить прозрачность процесса и обеспечить основу для дальнейшей автоматизации документооборота.

## Цель проекта

Разработать MVP учетной системы для автоматизации формирования счетов-фактур по доходным и НДС-проводкам на продажи услуг банка.

## Основные задачи

- прием проводок из АБС;
- обработка поступающих данных в режиме, приближенном к реальному времени;
- связывание проводок по заданным атрибутам;
- формирование проекта счета-фактуры на основе связанных проводок;
- просмотр и редактирование счетов-фактур пользователями;
- формирование реестров и отчетных данных;
- отправка счетов-фактур в другую учетную систему через очереди сообщений.

## Ожидаемый результат

Результатом проекта является минимально жизнеспособный продукт учетной системы, который позволяет автоматизировать базовый процесс формирования счетов-фактур и обеспечить дальнейшее развитие решения в промышленный сервис.
